nav - A terminal-based file navigator/viewer.

Invoked with:
    /path/to/nav_script && cd "$(cat /tmp/nav_dir.txt)"

Installation:
    Add an alias for nav to your ~/.bash_aliases file:
        alias f='/path/to/this_script && cd "$(cat /tmp/nav_dir.txt)"'

Controls:
    h : Go up a dir.
    j : Scroll down.
    k : Scroll up.
    l : Enter a dir.
